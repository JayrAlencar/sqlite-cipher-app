# basel-app
Base app for https://github.com/baseljs/basel.git

## Using
Install BASEL
```shell
npm install -g basel
```
Init the app
```shel
basel-init myApp -b tabs
```

Install dependencies
```shell
cd myApp && npm install
```

Run
```shell
npm start
```
